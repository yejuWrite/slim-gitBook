# 마크다운

<mark style="color:blue;">**마크다운(Markdown)**</mark>은 일반 텍스트에 서식을 추가할 수 있는 **경량  마크업 언어**다.

{% hint style="info" %}
**마크업 언어(Markup language)**

기호를  사용해 문서와 텍스트의 서식, 관계 및 구조를 나타내는 **기호 시스템**이다. &#x20;

마크업 언어의 예로는 HTML, Markdown,  XML 등이 있다.
{% endhint %}
